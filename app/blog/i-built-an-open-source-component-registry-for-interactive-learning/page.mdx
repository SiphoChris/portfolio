## What is lmscn?

lmscn is an open-source registry of production-ready interactive learning components built on shadcn/ui and Tailwind v4. You install them the same way you install any shadcn component â€” you own the code completely and can wire them into whatever backend or LMS logic you already have.

- **Match** â€” click-to-match term and definition pairs with mistake tracking

- **Scramble** â€” letter-tile word unscrambling with image and text clues

Each component is fully typed, accepts a single data prop, and fires an `onComplete` callback â€” so your application owns the logic, and the component handles the interaction.
